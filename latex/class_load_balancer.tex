\doxysubsection{Detailed Description}
Manages a pool of Web\+Servers and a queue of Requests. 

Responsibilities:\+
\begin{DoxyItemize}
\item Holds request queue
\item Assigns requests to idle servers
\item Advances time (clock cycles)
\item Generates new requests at random times
\item Dynamically scales servers to keep queue within desired range
\item Blocks specified IP ranges (simple firewall) 
\end{DoxyItemize}
