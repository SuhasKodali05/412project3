\doxysubsection{Detailed Description}
Reads key=value configuration from a text file. 

Supports IP block rules via:\+
\begin{DoxyItemize}
\item CIDR:\+ "{}192.\+168.\+0.\+0/\+24"{}
\item Range:\+ "{}10.\+0.\+0.\+1-\/10.\+0.\+0.\+200"{}
\item Single IP:\+ "{}8.\+8.\+8.\+8"{} 
\end{DoxyItemize}
