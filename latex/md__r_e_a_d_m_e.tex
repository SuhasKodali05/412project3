This is a C++17 simulation of a load-\/balancing system that:\+
\begin{DoxyItemize}
\item Generates random requests (IP in/\+out, time in cycles, job type P/\+S)
\item Uses a real queue ({\ttfamily std::\+queue\texorpdfstring{$<$}{<}\doxylink{struct_request}{Request}\texorpdfstring{$>$}{>}})
\item Processes work on a configurable number of {\ttfamily \doxylink{class_web_server}{Web\+Server}} instances
\item Dynamically adds/\+removes servers to keep the queue size between:\+
\begin{DoxyItemize}
\item {\bfseries{low\+\_\+queue\+\_\+per\+\_\+server \texorpdfstring{$\ast$}{*} servers}} and {\bfseries{high\+\_\+queue\+\_\+per\+\_\+server \texorpdfstring{$\ast$}{*} servers}}
\end{DoxyItemize}
\item Blocks configured IP ranges (CIDR or start-\/end ranges)
\item Logs colored events to console and writes a full log + summary to {\ttfamily lb.\+log}
