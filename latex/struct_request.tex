\doxysubsection{Detailed Description}
Represents a simulated web request. 

A \doxylink{struct_request}{Request} contains:\+
\begin{DoxyItemize}
\item IP in:\+ requester IP address (uint32)
\item IP out:\+ destination/\+result IP address (uint32)
\item time:\+ processing time in clock cycles (integer)
\item job\+Type:\+ \textquotesingle{}P\textquotesingle{} = Processing, \textquotesingle{}S\textquotesingle{} = Streaming 
\end{DoxyItemize}
